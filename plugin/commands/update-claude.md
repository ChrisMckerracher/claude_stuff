# Update CLAUDE.md

Add new conventions, patterns, or feedback to the project's CLAUDE.md file.

## Process

1. Read current CLAUDE.md
2. Identify appropriate section for new content
3. Add the new convention or feedback
4. Ensure it doesn't conflict with existing content
5. Format consistently with existing style

## What to Add

- New coding conventions discovered during implementation
- Patterns that should be followed project-wide
- Gotchas or pitfalls to avoid
- Tooling requirements
- Testing expectations

## Format Guidelines

- Use clear, imperative language
- Include examples where helpful
- Group related items together
- Keep entries concise
